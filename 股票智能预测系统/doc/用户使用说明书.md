# 股票智能预测系统 — 用户使用说明书

> 版本：v1.0 | 日期：2026-07-19

---

## 1. 系统简介

股票智能预测系统是一款基于 Transformer 深度学习模型的 A 股短期价格预测工具。用户只需输入股票名称，系统即可自动获取历史数据、训练模型，并生成未来 5 个交易日的收盘价预测，同时提供 DeepSeek 大模型驱动的专业投资分析报告。

### 1.1 核心功能

- 🔍 **智能搜索**：输入中文股票名称，自动匹配股票代码
- 📈 **价格预测**：基于 Transformer 模型预测未来 5 个交易日收盘价
- 🤖 **AI 分析**：DeepSeek 大模型生成专业投资分析报告
- ⭐ **收藏管理**：收藏关注的股票，一键触发分析
- 📊 **可视化**：ECharts 交互式图表，支持缩放、悬停详情

### 1.2 系统环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11、macOS、Linux |
| Python | 3.9 及以上 |
| 浏览器 | Chrome、Edge、Firefox（推荐 Chrome） |
| 网络 | 需连接互联网（获取行情数据和 AI 分析） |

## 3. 功能使用指南

### 3.1 股票预测

**操作步骤**：

1. 打开主页，在搜索框中输入 A 股中文名称（如"平安银行"、"贵州茅台"）
2. 从下拉列表中选择目标股票
3. 点击 **"🔍 开始预测"** 按钮（或按 Enter 键）
4. 等待约 15-30 秒，系统将完成以下步骤：
   - 获取最近 500 个交易日历史数据
   - 训练 Transformer 深度学习模型
   - 预测未来 5 个交易日收盘价
   - 调用 DeepSeek 大模型生成分析报告
5. 查看各模块展示的预测结果

**示例股票**：

| 名称 | 代码 | 交易所 |
|------|------|--------|
| 平安银行 | 000001 | 深圳 |
| 贵州茅台 | 600519 | 上海 |
| 比亚迪 | 002594 | 深圳 |
| 宁德时代 | 300750 | 深圳 |
| 中国平安 | 601318 | 上海 |

### 3.2 理解预测结果

#### 走势图

- **蓝色实线**：最近 60 个交易日的历史收盘价
- **紫色虚线**：未来 5 个交易日的预测收盘价
- **黄色虚线标注**：预测起点（当前最新交易日）
- 鼠标悬停可查看具体日期和价格

#### 预测表格

| 列 | 说明 |
|----|------|
| 日期 | 预测的目标交易日（格式：MM月DD日） |
| 预测收盘价 | 模型输出的预测值 |
| 较前一日变化 | 涨跌金额及百分比（▲涨 ▼跌） |

#### 模型信息

展示 Transformer 模型的完整参数：
- 架构参数：嵌入维度、注意力头数、编码器层数等
- 训练配置：优化器、学习率、损失函数等
- 模型复杂度：参数量（约 31.5 万）
- 数据统计：训练样本数、数据时间范围
- 训练指标：训练/验证损失、训练耗时

#### AI 分析报告

包含三个部分：
1. **公司经营业务概述**：该上市公司的主营业务和竞争格局
2. **近期财务数据与经营状况**：营收、利润、净息差等关键财务指标
3. **综合投资建议（含风险提示）**：投资评级、操作建议、风险提示

> ⚠️ **免责声明**：AI 分析仅供参考，不构成投资建议。股市有风险，投资需谨慎。

### 3.3 收藏管理

**查看收藏列表**：

点击导航栏的 **"⭐ 我的收藏"** 进入收藏页面。

**添加收藏**：

1. 在搜索框中输入股票名称
2. 从下拉列表中选择
3. 点击 **"➕ 添加收藏"**

**删除收藏**：

点击股票右侧的 **"🗑 删除"** 按钮，确认后移除。

**一键分析**：

点击股票右侧的 **"🔍 分析"** 按钮，系统自动跳转回主页并触发完整的预测+AI分析流程。

## 4. 常见问题

### Q1：输入股票名称后提示"未找到"？

请确保输入完整的 A 股中文名称（如"平安银行"而非"平安"），然后从下拉列表中选择。若仍无法找到，可能是该股票已退市或更名。

### Q2：预测耗时较长怎么办？

单次预测约需 15-30 秒，主要耗时在模型训练上。这是为保证预测准确性而设计的技术流程，请耐心等待。建议使用带 GPU 的机器可显著加速。

### Q3：AI 分析报告显示错误？

DeepSeek API 可能存在间歇性服务波动。若出现"服务暂时不可用"提示，请稍后重试。系统会自动保留已生成的预测图表和表格。

### Q4：收藏列表丢失了？

收藏数据存储在本地 SQLite 数据库中（位于 `.cache/favorites.db`）。请勿手动删除 `.cache` 目录。若确实丢失，系统会自动重建并恢复初始 5 支股票。

### Q5：如何切换深色/浅色主题？

当前版本仅支持深色主题，以保护用户视力并保持专业风格。

## 5. 技术指标说明

### 5.1 模型精度指标

| 指标 | 含义 |
|------|------|
| 训练损失 (MSE) | 模型在训练集上的均方误差，越低越好 |
| 验证损失 (MSE) | 模型在验证集上的均方误差，反映泛化能力 |
| 损失收敛 | 损失曲线下降后趋于平稳，说明模型已充分学习 |

### 5.2 图表交互

ECharts 图表支持以下交互操作：
- **悬停**：显示数据点详情
- **拖拽**：缩放查看局部数据
- **图例点击**：显示/隐藏特定数据系列
- **窗口缩放**：自适应窗口大小
