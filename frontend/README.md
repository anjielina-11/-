# 前端 (Vue 3)

## 技术栈
- Vue 3 + TypeScript + Vite
- Pinia (状态管理)
- Element Plus (UI 框架)
- ECharts (数据可视化)
- Axios (HTTP 请求)

## 功能模块
1. 用户登录/注册/个人信息
2. 农场和地块管理
3. 作物品种和种植档案
4. 病害图片上传和诊断结果展示
5. 农事日历和待办任务
6. 天气、病虫害、市场价格趋势图
7. 防治建议和规范原文对照
8. 模型监控面板（性能/漂移/未知样本）

## 启动
```bash
cd frontend
npm install
npm run dev
```
